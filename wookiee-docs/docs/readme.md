# wookiee-grpc 

## Install
wookiee-grpc is available for Scala 2.12 and 2.13. There are no plans to support scala 2.11 or lower.
```scala
libraryDependencies += "com.oracle.infy.wookiee" %% "wookiee-grpc" % "@VERSION@"
```
